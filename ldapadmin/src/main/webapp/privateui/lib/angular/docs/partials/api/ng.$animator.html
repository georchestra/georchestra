$animator (service in module ng )

Improve this doc

Description

The $animator.create service provides the DOM manipulation API which is decorated with animations.

Usage

$animator(scope, attr);

Parameters

ParamTypeDetails
scopeScope

the scope for the ng-animate.

attrAttributes

the attributes object which contains the ngAnimate key / value pair. (The attributes are passed into the linking function of the directive using the $animator.)

Returns

object

the animator object which contains the enter, leave, move, show, hide and animate methods.
